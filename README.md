# 🍀 로또 번호 추천기 (Lotto Generator)

현대적인 디자인과 사용자 편의성을 갖춘 웹 기반 로또 번호 생성기입니다.

## 🚀 주요 기능

-   **랜덤 번호 생성:** 1~45 사이의 중복 없는 6개 숫자를 생성하고 오름차순으로 정렬합니다.
-   **다크/라이트 모드:** 사용자의 선호에 따라 UI 테마를 실시간으로 변경할 수 있습니다.
-   **히스토리 관리:** 최근 생성한 번호 10개를 시간과 함께 기록하며, 브라우저 종료 후에도 유지됩니다.
-   **로또 공 디자인:** 실제 로또 번호 범위별 색상을 적용하여 시각적 즐거움을 제공합니다.

## 🛠 기술 스택

-   **HTML5**
-   **CSS3** (Custom Properties, Flexbox, Animations)
-   **JavaScript** (Vanilla JS, LocalStorage)

## 🎨 테마 설정
-   **Light Mode:** 깔끔하고 밝은 화이트톤 UI
-   **Dark Mode:** 눈이 편안한 다크그레이톤 UI

## 📦 설치 및 실행
이 프로젝트는 별도의 빌드 과정 없이 정적 파일로 구성되어 있습니다. `index.html` 파일을 브라우저에서 열기만 하면 바로 실행됩니다.
